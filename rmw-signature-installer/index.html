Download Instructions

  1. Click the download button below to download the RM Watson Email Signature Installer.
  2. Once downloaded, locate the file in your Downloads folder or where your browser saves downloaded files.
  3. Double-click the installer file to run it. You may need to approve security warnings from your operating system.
  4. After launching, the signature installer may take up to a minute to initialize - please be patient.
  5. Follow the on-screen instructions to find and install your personalized email signature.
Important: If the tool is blocked by Windows, select 'More info' then 'Run anyway.' The tool only works on Windows computers with Microsoft Outlook installed.
Download Signature Installer
After installation: You will need to set the signature as your default in Outlook. The installer will show you how to do this, but you may need to restart Outlook for the changes to take effect. In some cases, it can take up to 10 minutes for the signature to appear in Outlook's signature list.
